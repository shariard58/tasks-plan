## 📋 Per Task Process (প্রতিটা task এ এটা follow করো)

```
1. Requirements পড়ো carefully (5 min)
2. Concept research করো — MDN, javascript.info (15-30 min)
3. Approach ভাবো — pseudocode/rough plan (10 min)
4. Code লেখো — নিজে, copy-paste না (main work)
5. Test করো — edge cases, error scenarios (15 min)
6. Break করো — intentionally bug introduce করো, তারপর fix করো (10 min)
7. Refactor করো — cleaner, more efficient (10 min)
8. Git commit — proper message (5 min)
9. README লেখো — কী শিখলে, কোনটা কঠিন ছিল (10 min)
10. Teach it — কাউকে explain করো (বা rubber duck কে) (5 min)
```

## 🚫 Common Mistakes to Avoid

```
❌ Tutorial hell — 10টা tutorial দেখা ≠ JavaScript শেখা
❌ Copy-paste coding — পড়ো, বুঝো, নিজে লেখো
❌ Framework থেকে শুরু — আগে JS, তারপর framework
❌ শুধু syntax শেখা — syntax জানা ≠ JavaScript জানা
❌ "এটা কঠিন" বলে skip করা — কঠিন জিনিসই তোমাকে strong করবে
❌ শুধু পড়া, code না করা — পড়া 20%, code 80%
❌ AI দিয়ে সব করা — AI helper, AI teacher না
❌ Burnout — consistent 4-5 hours > burst 12 hours then crash
❌ একা শেখা — share করো, blog লেখো, community তে থাকো
```

## 🎯 When Stuck (আটকে গেলে)

```
1. Error message পড়ো carefully — 80% answer error message এই থাকে
2. console.log দিয়ে debug করো — কোথায় break হচ্ছে বের করো
3. MDN/javascript.info এ search করো
4. Chrome DevTools use করো (Sources, Console, Network, Memory)
5. Exact error message Google করো
6. 15 min break নাও — fresh mind এ try করো
7. Concept revisit করো — হয়তো foundation weak
8. Similar example দেখো (বুঝো, copy করো না)
9. Solve করার পর document করো — পরে আবার আটকাবে না
10. Community তে জিজ্ঞাসা করো (proper question format এ)
```
